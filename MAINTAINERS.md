## Responsibilities

- Review and merge pull requests once required CI checks pass.
- Triage issues and security reports (see [SECURITY.md](SECURITY.md)).
- Keep dependencies and the supported A2A / MCP SDK versions current.

## Becoming a maintainer

Open a discussion or issue proposing yourself after a track record of merged
contributions. Maintainers are added by an existing maintainer and mirrored to
the GitHub repository's permissions.
